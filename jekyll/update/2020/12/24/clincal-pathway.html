Clinical Pathway

Dec 24, 2020

Many years ago I came across a demo win32 application that is perfect for physicians. It is a clinical application with a unique method of entering clinical notes. Anyone who has attempted to write a clinical application knows that the hardest part of the process is data entry. Clinical data are made up of structured and unstructured information which is quite challenging to input. I couldn’t find or buy the application anywhere online but took a lot of notes on how the process work. It uses clinical pathway to enter data but most importantly it was easy and friendly. After many years and yet still couldn’t find any similar application I decided to create one. Unlike the original, this is a web version with many design changes to accomodate the web. However the idea remains the same.

Installation

The source is still in early development but operational.

  1. See laravel documentation for configuration and installation.
  2. Move the clinical_pathways folder in the configuration files folder to storage/app
  3. Run the cp.sql statement to restore the database.
